# Nexivo Website

Official website and update manifest for **Nexivo** - a unified game and app launcher for Windows.

## Contents

- `index.html` - Main website (hosted at https://nexivodock.co.uk)
- `version.json` - Update manifest for the Nexivo launcher app

## About Nexivo

Nexivo is a desktop application that:
- Automatically detects installed games from Steam, Roblox, Epic Games, and Rockstar Games
- Creates custom launch sessions that open multiple apps in sequence
- Tracks usage patterns and surfaces your most-used items
- Supports multiple color themes and customizable hotkeys
- Includes built-in update checking

## Features

⚡ **Smart Detection** - Automatically catalogs your installed games
🎮 **Custom Sessions** - Open multiple apps with one click
🎨 **Themed UI** - Choose from multiple color themes
⌨️ **Hotkey Toggle** - Show/hide with a custom global hotkey
📊 **Activity Tracking** - Intelligent usage-based sorting
🔄 **Auto-Updates** - Check for updates directly from the app

## Update Checking

Nexivo reads the `version.json` file to check for updates. Configure it in:
**Nexivo Settings → Updates**

- Manifest URL: `https://nexivodock.co.uk/version.json`
- Download Page: `https://nexivodock.co.uk/#downloads`
